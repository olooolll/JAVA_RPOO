## 1. Objective

This repository contains the solutions to the exercise lists assigned in the **Object-Oriented Programming** course.

The purpose of these activities is to reinforce the fundamental concepts of the object-oriented paradigm through practical implementation using the **Java** programming language.

---

## 2. Repository Organization

The exercises are organized into three lists, each containing 10 exercises:

- `lista01`: Exercises 01 to 10  
- `lista02`: Exercises 11 to 20  
- `lista03`: Exercises 21 to 30  

Each exercise is implemented in an independent Java file, named according to its corresponding number.

---

## 3. Project Structure

```

src/
│
├── lista01/
├── lista02/
└── lista03/

````

Each folder contains 10 Java files, totaling **30 exercises**.

---

## 4. Technologies Used

- **Language:** Java  
- **Compilation:** JDK  
- **Execution:** Terminal or IDE (Eclipse, IntelliJ, or VS Code)

---

## 5. Compilation and Execution

To compile an exercise:

```bash
javac Ex01.java
````

To execute:

```bash
java Ex01
```

> ⚠️ This process should be performed inside the corresponding folder of the exercise.
